# AI Prompt Vault 1.0

A professional repository for managing high-fidelity AI generation prompts.

## Features
- **Frontend**: Clean, dark-mode UI for high readability.
- **Backend**: LocalStorage integration for persistent data.
- **Tech Stack**: HTML5, CSS3, and Vanilla JavaScript.

## Project Focus
Currently optimized for **1970s cinematic realism** and **photorealistic portrait** generation.
