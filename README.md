# AI Prompt Vault v2.0

A high-fidelity technical library for managing complex AI generation strings. Designed for photographers, cinematic artists, and developers.

## 🚀 Features
- **Local Persistence**: Uses browser LocalStorage to keep prompts safe across sessions.
- **One-Click Copy**: Custom JavaScript logic to instantly copy complex strings to your clipboard with visual feedback.
- **Dynamic Asset Management**: Add and delete prompts in real-time to maintain a clean workflow.
- **Mobile Access**: Responsive UI designed for access via local network for on-the-go prompting.

## 📸 Specialized Collections
- **4K Photo Restoration**: Optimized strings for high-fidelity digital remastering.
- **Cinematic Realism**: 35mm film aesthetics and anamorphic lens prompts.
- **UI & Gaming**: Asset management for projects like EDISA'26 and mobile game development.

## 🛠️ Technical Stack
- HTML5 / CSS3 (Custom Dark Mode UI)
- JavaScript (Vanilla)
- Browser LocalStorage API
